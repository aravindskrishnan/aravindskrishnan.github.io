AUGMENTED REALITY

The cards have changed, and Augmented Reality is getting more real everyday. To ensure the smooth transition into the world of AR, we are offering you Hands on workshop on AR Technology. The workshop will be started off with basics of game development using Unity engine and then progress into Android studio and Vuforia SDK, for image tracking and 3D shooting using green mat. Introduction to C# programming for scripting player data and UI connection.
PREREQUISITES: Participants should bring there own laptop with the necessary softwares pre installed (which will be informed upon registration).
CONTACT: Aravind +91-7034267524

VIRTUAL REALITY

In future we will come across a time when we will find it difficult to separate the real world from the constructed one. Prepare yourself to embrace the change by joining hands on workshop on VR and game development. The workshop will start with focus on game development platform (unity) and further enhance the experience by introducing Java SDK, Google VR, Android Studio and C# scripting for player data and connecting to UI
PREREQUISITES: Participants should bring there own laptop with the necessary softwares pre installed (which will be informed upon registration).
